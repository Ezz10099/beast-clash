# Beast Clash — Game Design Base

## One-sentence pitch
A mobile 2D animal auto-battle game where each animal fights according to its real biological strengths.

## Battle format
- 3 vs 3 at MVP
- Side-view battlefield
- Animals auto-attack in speed order
- Player-controlled ultimates will be added later

## Animation philosophy
Do not start with full frame-by-frame fighting animation.
Use:
- sprite movement
- impact effects
- damage numbers
- camera shake
- status icons
- sound effects

## Animal design rules
Each animal should have:
- Role
- Trait list
- Basic attack
- Special/ultimate later
- Passive later
- Biome relation later

## MVP animal roster
1. Gorilla — bruiser/grappler/stun
2. Tiger — assassin/bleed
3. Crocodile — tank/bite/stun
4. Eagle — aerial striker
5. Snake — poison control
6. Rhino — charger tank

## Future tiers
- Common wildlife
- Apex animals
- Rare biome animals
- Extinct beasts
- Legendary bosses
