# Beast Clash

A mobile 2D animal auto-battle prototype where real animal traits become combat mechanics.

## Core idea
Animals are fighters. Their biology becomes gameplay:
- Tiger = fast bleed assassin
- Gorilla = heavy stun bruiser
- Crocodile = tank + bite lock
- Eagle = aerial striker
- Snake = poison control
- Rhino = charger tank

## How to run on mobile/SPCK
1. Put `phaser.min.js` in the project root beside `index.html`.
2. Open `index.html` in SPCK live preview / local server.
3. Press **Start Test Battle**.

## Current MVP
- 3v3 auto-battle
- placeholder animal sprites drawn by code
- HP bars
- turn-based attack loop
- basic skills/statuses: bleed, poison, stun
- simple battle animations using tweens
